# Weekly Tasks API

Контрольная работа по Express.js. Полноценное backend-приложение для таблицы дел на неделю. Приложение предоставляет REST API для управления задачами и простую статическую страницу, которая работает с этим API.

## Используемые технологии

- Node.js, Express 4
- Хранение данных в файле `data/tasks.json`
- Разделение логики на **routes** и **controllers**
- Собственный middleware-логгер запросов
- Раздача статических файлов через `express.static`

## Структура проекта

- `app.js` – основной файл приложения, настройка сервера, middleware, подключение маршрутов
- `routes/tasks.js` – маршруты `/api/tasks` (GET, POST, PUT, DELETE)
- `controllers/tasksController.js` – логика работы с задачами (чтение/запись `tasks.json`)
- `data/tasks.json` – данные задач на неделю
- `public/index.html` – простая страница «Таблица дел на неделю»
- `package.json` – зависимости и скрипты запуска

## Установка и запуск


Сервер запустится по адресу: `http://localhost:3000`.

## API эндпоинты

| Метод | URL                 | Описание                                      | Пример |
|------|---------------------|-----------------------------------------------|--------|
| GET  | `/api/tasks`        | Получить все задачи, с фильтрами по query     | `/api/tasks?day=monday` |
| GET  | `/api/tasks/:id`    | Получить задачу по идентификатору             | `/api/tasks/1` |
| POST | `/api/tasks`        | Создать новую задачу                          | body: `{ "title": "...", "day": "monday", "priority": "high" }` |
| PUT  | `/api/tasks/:id`    | Обновить задачу (например, изменить статус)   | body: `{ "completed": true }` |
| DELETE | `/api/tasks/:id`  | Удалить задачу                                | `/api/tasks/3` |

### Примеры использования

- Фильтрация по дню недели:  
  `GET /api/tasks?day=friday`
- Фильтрация по дню и статусу:  
  `GET /api/tasks?day=monday&completed=true`

## Статическая страница

Страница доступна по адресу: `http://localhost:3000/index.html`.

Функционал:

- Кнопка **«Загрузить все задачи»** делает запрос `GET /api/tasks` и выводит все задачи на неделю.
- Кнопка **«Только понедельник»** делает запрос `GET /api/tasks?day=monday` и показывает задачи только на понедельник.
